# NixOS-Conf

NixOS configuration; Dotfiles Hyprland

```bash
git clone https://github.com/StasLavr/NixOS-Conf.git
mv nixos-config $HOME/nix # Config is supposed to be in the ~/nix directory
cd $HOME/nix
sudo nixos-rebuild switch --flake .#NixOS
```
